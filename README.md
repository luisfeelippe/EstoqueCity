# Estoque City - Sistema de Controle e Gestão de Estoque


Eu gostaria de compartilhar com vocês um sistema de estoque que eu desenvolvi. A ideia do sistema é simples: adicionar e gerenciar clientes, produtos e vendas. O sistema ainda gera um relatório com as informações que são adicionadas no sistema.

A quem interessar:

- O sistema foi construído utilizando Bootstrap, HTML/CSS, JS, PHP e MySQL.

### Tela de login:

com inputs de ID e Senha, com opção para criação de novos usuários com campos de Nome e Senha.

### Dashboard:

Dashboard intuitivo que exibe informações essenciais como nome do usuário e sessão, número total de vendas registradas, valor total das vendas, número de clientes e número de produtos em estoque. Também exibe duas tabelas: produtos em falta e últimas 5 vendas.

![004 - Dashboard](https://github.com/user-attachments/assets/1df4baa9-4cf8-402b-bf53-51b2c3346d58)


### Tela de produtos 

tela de produtos com duas tabelas: uma para produtos em falta (estoque menor que 10) e outra para produtos em estoque. Opções de edição e exclusão de produtos, com modais para facilitar as ações. Produtos podem ser cadastrados através de um formulário simples.
